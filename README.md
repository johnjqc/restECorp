# RestStore

This Rest web service, generate a solution for the next problem:

1. El primer día de mayo se generó 1 tarjeta de crédito falsa desde la dirección 181.75.48.201.

2. El segundo día de mayo se generaron 4 tarjetas de crédito desde la dirección 181.75.48.202.

3. El tercer día de mayo se generaron 18 tarjetas de crédito desde la dirección 181.75.48.203.

4. El cuarto día de mayo se generaron 96  tarjetas de crédito desde la dirección 181.75.48.204.

5. El quinto día de mayo se generaron 600 tarjetas de crédito desde la dirección 181.75.48.205.

The solution is an endpoint that given a number day of month and generate the number of credit card.
